Cognify PHP 5 System
====================

Ready folders:
- admin/
- student/
- assets/
- inc/
- sql/
- uploads/questions/

Setup:
1. Create a database named cognify_db or import sql/cognify.sql in phpMyAdmin.
2. Update config.php database credentials if needed.
3. Put this folder inside htdocs, for example:
   C:\xampp\htdocs\cognify_php5\
4. Make sure uploads/questions/ is writable.
5. Open:
   http://localhost/cognify_php5/

Default admin:
- username: admin
- password: admin123

Included updates:
- Bloom-based categories only
- No difficulty level
- Dynamic radio choices
- Optional question image with required image label when image exists
- Admin-controlled quiz question limit
- Automatic Top 3 intelligence analysis
- Mint-green light responsive UI
